### 01장. 파이썬이란 무엇인가?
#### 01-5. 파이썬 둘러보기
* 사칙연산
  '''python
  1 + 2
  '''
  <code>
    1 + 2
    2 - 1
    3 / 2.4
    3 * 9
  </code>
* 변수 대입
  * 숫자 대입
    <code>
      a = 1
      b = 2
      a + b
    </code>
  * 문자 대입
    <code>
      a = "Python"
      print(a)
    </code>
* 조건문 if
  <code>
    a = 3
    if a > 1:
      print("a is greater than 1")
  </code>
* 반복문
  * for
    <code>
      for a in [1, 2, 3]:
        print(a)
    </code>
  * while
    <code>
      i = 0
      while i < 3:
        i = i + 1
        print(i)
    </code>
* 함수
  <code>
    def sum(a, b):
      return a+b
    print(sum(3,4))
  </code>
