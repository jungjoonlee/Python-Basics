### 01장. 파이썬이란 무엇인가?
#### 01-5. 파이썬 둘러보기
* 사칙연산
  ```python
  1 + 2
  1 - 2
  3 / 2.4
  3 * 9
  ```
* 변수 대입
  * 숫자 대입
    ```python
    a = 1
    b = 2
    a + b
    ```
  * 문자 대입
    ```python
    a = "Python"
    print(a)
    ```
* 조건문 if
  ```python
  a = 3
  if a > 1:
    print("a is greater than 1")
  ```
* 반복문
  * for
    ```python
    for a in [1, 2, 3]:
      print(a)
    ```
  * while
    ```python
    i = 0
    while i < 3:
      i = i + 1
      print(i)
    ```
* 함수
  ```python
  def sum(a, b):
    return a + b
  print(sum(3, 4))
  ```
